# Covid-in-Japan

### 対象OS: 
iOS 13.0~

### 開発環境/言語: 
Xcode/Swift

### 機能概要: 
- 日本の Covid に関する統計を表示
- 自身の健康チェックをカレンダーで管理
- 県別の統計を棒グラフ、円グラフで表示
- チャット機能(β版)

### 使用WebAPI:
[COVID-19 Japan Web API](https://github.com/ryo-ma/covid19-japan-web-api)
